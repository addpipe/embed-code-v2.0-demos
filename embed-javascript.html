Pipe Embed v2.0 JavaScript Demo

Made by the Pipe Video Recording Platform

This demo page embeds the Pipe Video Recording Client using the 2.0 JS embed code.

The recording client is controlled through a custom HTML UI linked to the recorder through the JavaScript events and control API. The recorder's default UI is hidden.

You can study the source code on GitHub.
